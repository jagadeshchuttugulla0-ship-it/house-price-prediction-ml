🏠 House Price Prediction Using Machine Learning

Project Overview

Built a machine learning model to predict house prices using the Ames Housing Dataset. Performed data preprocessing, feature engineering, model training and evaluation.

Dataset Information

  • Training Records: 1460
  • Test Records: 1459
  • Features: 80+

Models Used

  • Linear Regression
  • Random Forest
  • XGBoost

Results

  • Linear Regression R² = 0.8867
  • Random Forest R² = 0.8862
  • XGBoost R² = 0.9140
  • RMSE = 25,685
