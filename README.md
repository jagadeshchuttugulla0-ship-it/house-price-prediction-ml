# House Price Prediction using Machine Learning

## Overview

This project predicts house prices using the Ames Housing Dataset using Machine Learning techniques.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* XGBoost

## Dataset

* Training Records: 1460
* Test Records: 1459
* Features: 80+

## Models Used

* Linear Regression
* Random Forest
* XGBoost

## Results

| Model             | RMSE  | R² Score |
| ----------------- | ----- | -------- |
| Linear Regression | 29476 | 0.8867   |
| Random Forest     | 29539 | 0.8862   |
| XGBoost           | 25685 | 0.9140   |

## Best Model

XGBoost achieved:

* RMSE: 25,685
* R² Score: 91.4%
